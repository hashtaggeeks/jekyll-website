    Python lambda Functions

    Python lambda functions are anonymous in-line function consisting of a single expression which is evaluated when the function is called. Lambda functions are also referred to as lambda abstractions, a direct reference to the abstraction model of Alonzo Church’s original creation. Python is not inherently a functional language, but it adopted some functional concepts using map(), filter(), reduce(), and the lambda operator.

    A lambda function can’t contain any statements. In a lambda function, statements like return, pass, assert, or raise will raise a SyntaxError exception
